# Our Home

> A simple smart home dashboard

Built in Elixir, Phoenix, LiveView, and Tailwind CSS

## Getting started

Clone the project locally. Create a copy of the environmental variables:

```bash
cp .env.example .env
```

Edit `.env` and configure values.

```bash
vim .env
```

Get `mix` dependencies:

```bash
mix deps.get
```

Start the server:

```bash
bin/start
```
